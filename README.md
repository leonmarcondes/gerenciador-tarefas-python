# Gerenciador de Tarefas em Python

Aplicação simples desenvolvida em Python para gerenciamento de tarefas via terminal.

## Funcionalidades
- Adicionar tarefas
- Listar tarefas
- Marcar tarefas como concluídas
- Remover tarefas
- Armazenamento das tarefas em arquivo JSON

## Tecnologias utilizadas
- Python

## Como executar
1. Clone o repositório
2. Execute o arquivo `main.py` no terminal
